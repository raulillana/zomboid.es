# Ejecucion Local

## Requisitos

- Node.js 18 o superior
- npm

## Instalacion

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

El servidor estara disponible en `http://localhost:4321`

## Build

```bash
npm run build
```

Los archivos estaticos se generaran en la carpeta `dist/`

## Preview

Para previsualizar el build:

```bash
npm run preview
```
